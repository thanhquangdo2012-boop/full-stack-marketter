> ⚠️ **NỘI DUNG NHÁP** — khung chấm điểm dựng theo mô tả trong báo cáo ngách ("khung chấm cho tờ trình, content, hình ảnh"). Cần HV xem lại, tinh chỉnh tiêu chí theo đúng kinh nghiệm thật trước khi bán cho khách.

Trước khi gửi/nộp bất kỳ sản phẩm nào ngoài chuyên môn gốc, tự chấm theo thang 1-3 cho từng tiêu chí (1 = chưa đạt, 2 = tạm ổn, 3 = tốt). Dưới 2 điểm ở bất kỳ mục nào — nên làm lại trước khi gửi.

## Tờ trình / báo cáo
| Tiêu chí | 1 | 2 | 3 |
|---|---|---|---|
| Có kết luận/đề xuất ngay từ đầu, không phải đọc hết mới hiểu ý chính | | | |
| Có số liệu cụ thể, không chỉ có cảm nhận chung chung | | | |
| Không dài quá mức cần thiết (dưới 1 trang cho việc thường ngày) | | | |
| Có ít nhất 1 phương án thay thế, không ép người đọc chỉ 1 lựa chọn | | | |

## Content / bài viết
| Tiêu chí | 1 | 2 | 3 |
|---|---|---|---|
| Đúng đối tượng đọc mục tiêu, không viết chung chung cho "mọi người" | | | |
| Có 1 thông điệp chính rõ ràng, không lan man nhiều ý | | | |
| Câu mở đầu đủ hấp dẫn để người đọc không thoát ra ngay | | | |
| Không có lỗi chính tả/ngữ pháp cơ bản | | | |

## Hình ảnh / thiết kế (khi không phải chuyên môn gốc)
| Tiêu chí | 1 | 2 | 3 |
|---|---|---|---|
| Đúng brand guideline (màu, font, logo) nếu công ty có sẵn | | | |
| Truyền tải đúng thông điệp chính, không chỉ đẹp mà lạc đề | | | |
| Dễ đọc ở kích thước thực tế sẽ hiển thị (điện thoại, banner...) | | | |

## Cách dùng
Chấm xong, nếu có mục dưới 2 điểm: quay lại đúng phần tương ứng trong [Danh sách nguồn học](nguon-hoc-chon-loc.md), sửa lại, rồi chấm lại trước khi gửi.
